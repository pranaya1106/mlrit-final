Course Outcomes Description Booklets
Computer Science & Engineering — MLRIT, Dundigal
R25 Regulation — Year I
Semester I & II Core Courses
Programming for Problem Solving (C)Sem I
CO1Understand the structure and programming constructs of C language.
CO2Apply control flow statements and functions to solve problems.
CO3Use arrays, strings, and pointers effectively in programs.
CO4Implement structures, unions, and file I/O operations.
CO5Develop modular programs for real-world problem solving.
Data StructuresSem II
CO1Understand linear data structures: arrays, stacks, queues, and linked lists.
CO2Implement tree data structures (BST, AVL, Heaps).
CO3Apply graph algorithms (BFS, DFS, shortest paths).
CO4Analyze time and space complexity of algorithms.
CO5Select appropriate data structures for given problems.
R25 Regulation — Year II
Core Courses — Semesters III & IV
Database Management SystemsSem III
CO1Understand relational model, ER diagrams, and normalization.
CO2Write SQL queries including joins, aggregates, and subqueries.
CO3Design and implement relational database schemas.
CO4Understand transaction management, concurrency, and recovery.
CO5Apply indexing, hashing, and query optimization techniques.
Operating SystemsSem IV
CO1Understand OS concepts: processes, threads, and scheduling.
CO2Apply process synchronization and deadlock handling techniques.
CO3Understand memory management: paging, segmentation, virtual memory.
CO4Understand file systems, I/O management, and disk scheduling.
CO5Analyze security and protection mechanisms in operating systems.
R25 Regulation — Year III & IV
Core & Elective Courses
Machine LearningSem V
CO1Understand supervised and unsupervised learning algorithms.
CO2Apply regression, classification, and clustering techniques.
CO3Implement neural networks and deep learning frameworks.
CO4Evaluate model performance using appropriate metrics.
CO5Develop ML solutions for real-world datasets and problems.
Computer NetworksSem V
CO1Understand network architecture, OSI and TCP/IP models.
CO2Apply data link protocols, error detection, and MAC techniques.
CO3Understand routing algorithms and IP addressing.
CO4Analyze transport layer protocols (TCP, UDP) and congestion control.
CO5Understand application layer protocols and network security basics.
Cryptography & Network SecuritySem VI
CO1Understand symmetric and asymmetric encryption algorithms.
CO2Apply hash functions, digital signatures, and PKI.
CO3Understand network security protocols (SSL/TLS, IPSec, SSH).
CO4Analyze common attacks and apply countermeasures.
CO5Design secure systems using cryptographic best practices.
M.Tech — Year I
Core Courses — Semesters I & II
Advanced Data Structures & AlgorithmsSem I
CO1Analyze advanced data structures: B-trees, Red-Black trees, Skip lists.
CO2Apply algorithm design paradigms: DP, greedy, divide and conquer.
CO3Evaluate algorithm complexity using formal analysis techniques.
CO4Implement and analyze graph algorithms for complex network problems.
CO5Apply advanced algorithmic techniques to research-level problems.
Machine Learning and Deep LearningSem I
CO1Apply advanced ML algorithms including SVM, Random Forests, and ensembles.
CO2Design and train deep neural networks (CNN, RNN, LSTM, Transformers).
CO3Apply transfer learning and fine-tuning strategies.
CO4Evaluate and optimize models for real-world deployment.
CO5Conduct original research integrating ML/DL solutions to domain problems.
Research Methodology & Technical WritingSem II
CO1Understand research methodologies and scientific writing conventions.
CO2Critically review and synthesize literature from reputed journals.
CO3Formulate research questions and design experiments.
CO4Write well-structured research papers for journal/conference submission.
CO5Apply ethical norms in research and handle data responsibly.
